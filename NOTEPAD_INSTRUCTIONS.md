# Notepad Instructions

## Overview
This is a disguised communication system that looks like a regular notepad but allows private messaging through WhatsApp.

## Setup
1. Select a group from the dropdown (Config ID)
2. The group must have you, the other person, and the NeuraX bot

## Commands (Type and press Ctrl+Enter)

### Send Message
```
::send Hello, how are you?
```
- Sends "Hello, how are you?" to the WhatsApp group via bot

### Remove Text in Angle Brackets
```
::send Hi what's<ignore this> up?
```
- Sends: "Hi what's up?" (text in `<>` is removed)

### View History
```
::Hist
```
- Shows last 10 messages (yours and theirs)

### Clear All
```
::clear
```
- Clears all notepad content and history

## Receiving Messages
When the other person sends a message in WhatsApp using `@prvt <message>`:
- It appears in your notepad disguised in a technical paragraph
- Example: "The system architecture demonstrates hello there which provides excellent scalability."
- The actual message is hidden in the middle

## Important
- Press **Ctrl+Enter** to execute commands (not just Enter)
- Messages are disguised to look like technical documentation
- No one can tell this is a messaging system
